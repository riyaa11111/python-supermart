# 🛒 Python SuperMart

An interactive, lightweight e-commerce cart and checkout application built using Python and Streamlit. This project transitions standard terminal-based retail logic into a functional, user-friendly web interface.

## ✨ Features
- **Dynamic Product Catalog:** View real-time pricing grids for stock keeping units (SKUs).
- **Stateful Shopping Cart:** Persists selected quantities dynamically throughout the session utilizing Streamlit State Management.
- **Automated Pricing & Discount Matrix:** Calculates tiered pricing models automatically (5% off orders ≥ ₹200, 10% off orders ≥ ₹500).
- **Responsive Web GUI:** Fully interactive web design with built-in form validations without requiring HTML/CSS/JS.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Framework:** Streamlit (UI/UX deployment)
